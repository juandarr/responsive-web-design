Basics

The CSS grid is a new standard that makes easy to create complex responsive layouts. It works by turning HTML elements into a grid, and lets you place child element anywhere within.

You can define an CSS grid by using the property and value display: grid;. The parent element is referred to as the container while the children are called items.

Defining the display property as grid doesn't state the structure of the grid. To state a structure, you use the properties like grid-template-columns and grid-template-rows. These properties can have one or more values representing a new column/row as well as its width/height.

Using only one of the two properties (grid-template-rows or grid-template-columns) will set the other automatically. By using both properties you can clearly state the dimensions of the grid.

Grid units

The set the size of each dimension (width or height) you can use absolute (`px`) or relative (`em`) units. You can also use the following ones:

  • fr : sets the column or row to a fraction of the available space.
  • auto : sets the columns or row to the width or height of its content automatically.
  • % : adjust the column or row to the percent width of its container.
Gaps and spacing

When you want to create a gap between the columns you use the property `grid-column-gap` with the value of the gap in absolute or relative size units. Gaps between rows can be added with the property `grid-row-gap`, and the value of the gap in absolute or relative units.

grid-gap is a shorthand property for grid-row-gap and grid-column-gap`. If one value is set it will create gaps for both rows and columns. If two values are set, the first one sets the row gap and second one sets the column gap.

A useful concept here is the one of line. A grid is represented by horizontal and vertical lines numbered from 1 up, with the origin being the top left corner of the grid. With grid-column you can control how many columns an item will consume. For example, the value `1/3` is used for the item to include columns 1 and 2. You can also define how many rows an item spans from by using the property grid-row

Alignment

In CSS grid the content of each item is located in a box which is referred to as a cell. You can align the content's position within its cell horizontally by using the justify-self property on a grid item.

By default, this property's value is `stretch`. Here are other options:

  • start : aligns the content at the left of the cell.
  • center : aligns the content in the center of the cell.
  • end : aligns the content at the right of the cell

You can also align an item vertically in its cell using the align-self property. It receives the same values justify-self receives.

When you want all the items from a CSS grid to share the same alignment, you can use justify-items. It receives all the values described in the previous lessons: stretch as default, center, start and end.You can set the vertical alignment for all the items in a grid using align-items.

Areas

You can group cells of your grid together into an area and give the area a custom name. You can do this using grid-template-areas on the container.

.container { grid-template-areas: "header header header" "advert . content" "footer footer footer"; }

Each pair of quotes define a row. Each word defines a cell in the grid and you use the same label to put them together in the same "label" group - In the example above, first three cells belong to the "header" group. Fourth cell belong to the area "advert", fifth designate an empty cell in the grid, sixth belong to the "content" area. Last three cells belong to the "footer" area.

After the creation of a grid template you can place an item in your custom area by referencing the name you gave in the property grid-area of the item.

There is another way to use the grid-area property. If your grid doesn't have an areas template to reference, you can create an area on the fly for an item to be placed as follows: grid-area: 1/1/2/4;. The instruction is read as follow: grid-area: horizontal line start/vertical line start/horizontal line end/ vertical line end. In the previous line, grid-area will cover horizontal lines from 1 to 2, and vertical lines from 1 to 4.

Repeat

The repeat function can be used to repeat the definition of a row of column/row (or a set of them) n times. You can also mix the repeat function with literal definitions.

Minmax

Another function used in grid templates is minmax. This function allows to set a size range so the item resizes when the grid container dimensions change.

Responsive and flexible layout

You can use auto-fill with the repeat function to create as many columns/rows as required depending on the size of the container. This can be combined with minmax to create flexible layouts:

repeat(auto-fill, minmax(60px, 1fr));

auto-fit works similar to auto-fill. The only difference is that when the container's size exceeds the size of all the items combined, auto-fill keeps inserting empty rows or columns and pushed your items to the side. auto-fit collapses those empty rows/columns and stretches your items to fit the size of the container.

Media queries combined with the CSS grid features are powerful tools to create responsive design.

Turning an element into a grid affect the behavior of its direct descendants. By turning a direct descendant of a grid element into a grid, you have a grid within a grid.
